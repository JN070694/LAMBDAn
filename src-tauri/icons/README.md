# Icons

Place your app icons here. Tauri requires:

- icon.png (512x512 recommended)
- icon.ico (Windows)
- icon.icns (macOS)

You can generate all variants from a single PNG using:

  cargo tauri icon icon.png

This will auto-generate all required sizes into this folder.

The LAMBDAn logo spec:
- White background
- Black rounded-square border
- Uppercase Λ (Georgia serif, large, clipped)
- Lowercase n superscript upper-right
